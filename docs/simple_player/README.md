# Simple Player

This is the most basic implementation of music-manager.  
All it does is create the music-manager object, and subscribes to a few events.  
What it can do:  
- Play/Pause
- Fast Forward/Rewind
- Next/Last Track
- Next/Last Folder
- Volume Up/Volume Down
